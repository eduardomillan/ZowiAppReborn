# Android versions support

As of now (July 2026), the Android versions with active support are **14, 15, 16, and 17**. All earlier versions have reached end of life and no longer receive security patches.

Here is a detailed summary:

| Version | Support Status | End of Support Date (estimated) |
| :--- | :--- | :--- |
| **Android 17** | **In development / Beta** | To be determined |
| **Android 16** | **Supported** | To be determined |
| **Android 15** | **Supported** | To be determined |
| **Android 14** | **Supported** | To be determined |
| Android 13 | Unsupported | March 2026 |
| Android 12 / 12.1 | Unsupported | March 2025 |
| Android 11 | Unsupported | February 2024 |
| Android 10 or earlier | Unsupported | Varies (all ended) |

### Things to keep in mind about support

*   **What "unsupported" means**: Devices that cannot be upgraded to a supported version no longer receive critical security patches, which makes them more vulnerable to threats.
*   **The manufacturer factor**: Google provides the base code, but manufacturers (Samsung, Xiaomi, etc.) customize and distribute updates for their devices. This means actual support can vary significantly depending on brand and model.

If you want to check the support status of a specific model, the best option is to consult the manufacturer's support page directly.
